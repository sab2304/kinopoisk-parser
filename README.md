# Парсер Кинопоиска

Простой и эффективный парсер для сбора информации о фильмах с сайта Кинопоиск.

##  Возможности

- Сбор структурированных данных о фильмах
- Поиск фильмов по названию  
- Экспорт в JSON, CSV, TXT форматы
- Готовность к интеграции в рекомендательные системы

```python
# Запуск упрощенной версии
