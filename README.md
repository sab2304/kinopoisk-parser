# Парсер Кинопоиска

## Постановка задачи

**Задача**: Реализовать парсер для сбора структурированной информации о фильмах с сайта Кинопоиск.

**Обоснование**: Собранные данные могут быть использованы для построения улучшенной рекомендательной системы фильмов, которая будет учитывать данные из различных источников. Парсер решает задачу извлечения и структурирования информации о фильмах для последующего анализа и машинного обучения.

**Основная функция**: Парсер принимает URL фильма на Кинопоиске и возвращает структурированную информацию о фильме, включая рейтинг, жанр, актерский состав и другую информацию.

## Возможности

- Поиск фильмов по названию
- Получение детальной информации о фильме:
  - Название и оригинальное название
  - Год выпуска
  - Рейтинг Кинопоиска и количество оценок
  - Жанры
  - Страны производства
  - Режиссеры
  - Актерский состав
  - Продолжительность
  - Описание
- Экспорт данных в различные форматы:
  - JSON
  - CSV
  - TXT
